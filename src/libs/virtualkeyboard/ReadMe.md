枚举插入表达式的各种情况：

按键内容    	光标位置      			操作内容    	操作方式				操作后光标的位置
数字		数字左/中/右/空运算符    	数字		插入当前光标处				数字后面
数字		非数字左              		数字+空格 	插入当前光标处				数字后面
数字		非数字中      			空格+数字	跳过当前非数字项,插入后面		数字后面
数字		非数字右      			空格+数字	插入当前光标处				数字后面

小数点    	数字左/空运算符          	"0."        	当前项不包含小数点，插入当前光标处	在"."后面
小数点    	数字中/右              		"."        	当前项不包含小数点，插入当前光标处	在"."后面
小数点    	非数字左              		"0."+空格     	当前项非数字，插入当前光标处		在"."后面
小数点    	非数字中              		空格+"0."      	跳过当前非数字项,插入后面		在"."后面
小数点    	非数字右              		空格+"0."      	插入当前光标处				在"."后面

非数字		空运算符			非数字		插入当前光标处				非数字后面
非数字		数字左/非数字左			非数字+空格	插入当前光标处				非数字后面
非数字		数字中/非数字中			空格+非数字	跳过当前操作项,插入后面			非数字后面
非数字		数字右/非数字右			空格+非数字	插入当前光标处				非数字后面

退格		空运算符			无		不做处理				不做处理
退格		数字左/非数字左(首项)		无		不做处理				不做处理
退格		数字左/非数字左(非首项)		无		不做处理				将光标向左移动一个单位
退格		数字中				无		删除当前光标左侧一个数字/小数点		将光标向左移动一个单位
退格		数字右(仅1位&&首项)		无		删除当前光标左侧一个数字/小数点+空格	将光标向左移动一个单位
退格		数字右(仅1位&&仅首项)		无		删除当前光标左侧一个数字/小数点		将光标向左移动一个单位
退格		数字右(仅1位&&非首项)		无		删除空格+当前光标左侧一个数字/小数点	将光标向左移动两个单位
退格		数字右(>1位)			无		删除当前光标左侧一个数字/小数点		将光标向左移动一个单位
退格		非数字中/右(首项)		无		删除当前的非数字项+空格			将光标移动到当前非数字项左侧的位置
退格		非数字中/右(仅首项)		无		删除当前的非数字项			将光标移动到当前非数字项左侧的位置
退格		非数字中/右(非首项)		无		删除空格+当前的非数字项			将光标移动到当前非数字项左侧的位置 - 1
退格		参数内				无		删除光标处的参数			将光标移动到当前参数的左侧位置
退格		函数内				无		删除光标处的函数			将光标移动到当前函数的左侧位置
退格		参数函数内			无		删除光标处的函数			将光标移动到当前函数的左侧位置

函数 		空运算符			函数体		插入当前光标处				将光标移动到第一个参数的末尾
函数		参数内				参数,函数	删除光标处的参数，插入函数		将光标移动到该插入函数的第一个参数的末尾
函数		参数函数内			参数,参数函数	删除光标处的参数函数，插入函数		将光标移动到该插入函数的第一个参数的末尾
参数		参数内				参数		插入当前光标处				将光标移动到该插入参数的末尾
参数		参数函数内			参数,参数函数	删除光标处的参数函数，插入参数		将光标移动到该插入参数的末尾

空格		空运算符
空格		数字左
空格		数字中
空格		数字右
空格		非数字左
空格		非数字中
空格		非数字右
